## Retail Sales Data Pipeline

Built a data pipeline using Python and SQL to clean, transform, and analyze retail sales data.

### Tools Used
- Python (pandas)
- SQL (SQLite)
- Google Colab

### What I Did
- Cleaned raw transactional data
- Created new metric: total_sales
- Loaded data into SQL database
- Analyzed sales by category, product, and region

### Key Insight
Identified top-performing categories and products driving revenue.

### Dashboard

Built an interactive Power BI dashboard to visualize:

Sales by category and region
Top-performing products
Profit by customer segment

Includes slicers for dynamic filtering.
